进程的用户地址空间都是独立的。通信必须通过内核。  

### 管道

```C
$ ps auxf | grep mysql
```
上面的竖线就是管道。  
管道的意思是将前一个命令的输出作为后一个命令的输入。
因此相互通信需要两个管道。
竖线是匿名管道，用完即销毁。

命名管道的创建：
```
$ mkfifo myPipe
```
Linux理念：**一切皆文件**

```C++
echo "hello" > myPipe // 写入数据到管道

cat < myPipe // 读取管道里的数据
```
